# Air Hockey – Unity

Projeto desenvolvido na Unity como atividade de laboratório, consistindo na criação de um jogo simples de Air Hockey.

## Requisitos do Projeto

- Players posicionados na vertical (um na parte superior e outro na parte inferior).

- IA básica controlando o player superior.

- Mouse controlando o player inferior.

- Movimentação livre em qualquer direção, sem invadir o campo adversário.

- Imagem de fundo aplicada à cena.

- Palhetas com formato redondo.

- Gols com tamanho reduzido (não ocupam toda a parede) e com cores distintas.

- Sons reproduzidos nas colisões.
